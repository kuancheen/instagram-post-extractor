Instagram Post Xtractor is a premium web application that allows users to extract post descriptions and perform OCR (Optical Character Recognition) on images from public Instagram posts and reels.

## Purpose
Built to simplify the process of gathering textual information from social media posts, handling CORS constraints, and providing a high-fidelity user experience.

## Features
- **Meta Extraction**: Extracts `og:description` and `og:image` using robust proxy fallbacks.
- **OCR Engine**: Powerful text extraction from images powered by Tesseract.js.
- **Premium Design**: Glassmorphic UI with dark mode and smooth transitions.
- **Reliability**: Automatic retries and proxy switching for improved uptime.
- **Responsive**: Fully functional on both desktop and mobile devices.

## Installation & Setup
Since this is a static web application, no installation is required.
1. Clone the repository.
2. Open `index.html` in any modern web browser.
